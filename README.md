# 🌍 Language Detector – Real-Time Language Detection App

Language Detector is a modern and interactive web app designed to identify the language of any given text. With a clean interface, confidence scoring, and support for **22 languages**, it transforms simple text input into actionable insights.  

Built with a focus on usability, performance, and real-world deployment, this project demonstrates advanced **Machine Learning**, **NLP**, and **web app deployment** concepts.

---

## ✨ Key Features

- 🌐 Detects **22 languages** including English, Hindi, Tamil, French, Spanish, German, Italian, Portuguese, Dutch, Russian, Chinese, Japanese, Korean, and more  
- 📊 Displays **prediction confidence** to measure reliability  
- ⚠️ Warns users if the language is **not recognized or confidence is low**  
- 🖥️ Web-based, **interactive interface** via Streamlit  
- ❤️ Real-time predictions with **ML model hosted on Hugging Face**  
- 📌 Vectorizer hosted on **GitHub** for preprocessing  
- 📱 Fully responsive for desktop and mobile use  
- 💻 Simple, clean, and professional UI  

---

## 🛠️ Tech Stack

- **Python** – Core programming language  
- **Streamlit** – Web app framework for interactive UI  
- **Scikit-learn** – ML model & vectorizer  
- **Joblib** – Model serialization and loading  
- **Hugging Face Hub** – Model hosting  
- **NumPy** – Numerical operations
